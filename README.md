# 🎧 TTS 转发服务器 (TTS Forwarder)

这是一个基于 Android 原生平台开发的**高效本地 TTS HTTP 转发服务器**。它可以将系统中已安装的任何 TTS (文字转语音) 引擎转换为标准的 HTTP 音频流接口，便于**阅读 (Legado)** 等第三方阅读软件调用本地 TTS 进行朗读。

---

## ✨ 核心功能

*   **引擎选择**：支持一键切换并锁定系统内已安装的各类 TTS 引擎。
*   **网络转发**：支持自定义端口监听，极速流式输出 `.wav` 音频。
*   **阅读对接**：支持一键导入配置至阅读 (Legado) 软件，或快速复制 JSON 配置。
*   **实时日志**：实时展示请求日志，界面自动滚动到最新记录，便于调试。
*   **后台防杀**：一键设置电池优化白名单，确保持续稳定的后台监听。
*   **缓存清理**：深度扫描并一键清理合成产生的临时音频缓存。

---

## 🛠️ 技术架构

*   **开发语言**：Kotlin
*   **UI 框架**：Jetpack Compose (Material 3)
*   **本地存储**：Room Database (用于存储请求日志)
*   **后台机制**：Android Foreground Service (前台服务)
*   **并发处理**：Kotlin Coroutines & Flow
