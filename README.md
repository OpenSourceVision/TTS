# 🎧 TTS 转发服务器 (TTS Forwarder)

这是一个基于 Android 原生平台开发的**高效本地 TTS HTTP 转发服务器**。它可以将系统中已安装的任何 TTS (文字转语音) 引擎，转换为标准的本地/局域网 HTTP 音频流接口，从而让**阅读 (Legado)** 等第三方小说阅读软件能够极速、顺畅地调用系统 TTS 语音合成，享受最优质、稳定的本地化朗读体验。

---

## ✨ 核心功能亮点

### 1. 🎛️ 引擎选择 (TTS Engine Selection)
*   **多引擎兼容**：智能识别系统内已安装的所有 TTS 语音合成引擎（如 Google 语音服务、微软 TTS、小米/华为系统 TTS 等）。
*   **一键切换**：直观的下拉菜单，允许用户快速测试并锁定目标合成引擎。

### 2. 🔌 端口监听与网络转发 (Listening Ports)
*   **自定义端口**：支持修改本地 HTTP 服务器监听端口，完美避开端口冲突。
*   **极速流式输出**：接收来自客户端的语音合成请求，利用协程在后台快速生成高品质的 `.wav` 音频并实现流式传输。

### 3. 📖 阅读 APP 一键对接 (Legado Easy Integration)
*   **一键配置导入**：支持一键将接口配置直接导入到**阅读 (Legado)** 软件中，免去繁琐的手动配置过程。
*   **剪贴板快速复制**：内置标准的 Legado TTS 引擎 JSON 配置，一键复制到剪贴板，方便在其他设备或阅读器中使用。

### 4. 📄 实时请求日志 (Real-time Logging Engine)
*   **实时交互跟踪**：详细展示每次 TTS 合成请求的文本、来源、引擎名称、字符数和时间戳。
*   **人性化滚动优化**：自动对齐并定位到**最新生成的日志（最底部）**，无需手动向下滑动，交互极为顺畅。

### 5. 🔋 电池优化白名单 (Battery Whitelist)
*   **后台防杀**：一键引导进入系统电池优化设置，免受 Android 系统在后台冻结或杀死 HTTP 服务的干扰。
*   **智能状态感知**：当应用成功加入白名单（无限制运行）后，**去设置/加入白名单按钮将自动隐藏**，保证界面清爽。

### 6. 🧹 深度缓存清理 (Smart Cache Cleaner)
*   **多级缓存扫描**：深度扫描应用内部缓存及外部临时音频文件夹，展示精确的占用空间（B、KB、MB）。
*   **彻底释放空间**：一键清除所有合成的临时音频缓存，避免产生垃圾文件，确保手机存储空间充足。

## 🛠️ 技术架构

*   **开发语言**：Kotlin 100%
*   **界面技术**：Jetpack Compose (Material Design 3 风格)
*   **本地数据库**：Room Database (用于安全地持久化记录合成历史与日志)
*   **后台服务**：Android Foreground Service (前台服务，配合通知栏提供稳健的持续监听服务)
*   **并发处理**：Kotlin Coroutines & Flow (高并发非阻塞数据传输)

---

## 🚀 编译与运行

1. 克隆或导入本项目到 **Android Studio**。
2. 确保 Gradle 使用兼容的 JDK。
3. 连接您的 Android 设备或模拟器。
4. 运行 `gradle :app:assembleDebug` 生成安装包，或点击运行按钮直接部署到设备。
