# 🎧 Android TTS 转发服务器 (TTS HTTP Forwarder)

将 Android 手机本地的 TTS (Text-to-Speech) 语音引擎转换为标准的 **HTTP 音频流接口**，供兼容的第三方阅读或文本朗读客户端进行流式 `.wav` 语音合成。

---

## ✨ 核心特色

*   **极速流式响应**：点击朗读后立即开始播放，声音秒开不卡顿，支持随时打断并快速跟进。
*   **智能发音纠正**：支持多音字纠错与自定义文本替换，精准修正小说角色的错误发音。内置智能优化算法，无论配置多少规则都不会增加耗电。
*   **防错自动清洗**：在配置替换规则时，就算不小心多输入了空格、回车或多余的分隔符，系统也会自动完成净化，防止规则配置错误导致朗读异常。
*   **稳定防崩保护**：遇到大量连续朗读请求时自动平滑排队处理，智能拦截垃圾超长文本请求，确保听书过程中绝不出现爆音、破音或闪退卡死。
*   **丝滑后台听书**：
    *   **稳定保活不中断**：针对现代手机系统进行了深度优化，后台锁屏持续朗读不中断、不断连。
    *   **热开关防冲突**：即使频繁快速点击启动/关闭开关，系统也会自动清理上一轮的连接，彻底避免端口冲突与死锁。
*   **规则安全保存**：应用更新升级时，您辛苦配置的所有发音规则、系统设置和历史记录都会完整无损保留，绝不丢失。
*   **省心省电辅助**：一键引导加入手机电池白名单，保障锁屏稳定朗读；内置音频缓存自动扫描，支持一键深度清理，绝不占用多余的手机存储空间。

## 🔧 快速上手

### 1. 启动服务
在 App 主界面打开 **「启动服务」** 开关，即可看到当前的服务器监听地址（例如：`http://127.0.0.1:8080/tts`）。

### 2. 对接外部客户端
在兼容的第三方客户端中，添加如下标准网络语音引擎配置（以标准 JSON 格式为例）：

```json
{
  "id": "local_tts_forwarder",
  "name": "本地TTS转发",
  "url": "http://127.0.0.1:8080/tts?text={speakText}",
  "contentType": "audio/wav",
  "concurrentTasks": 1
}
```

## 🛠️ 技术开发

*   **界面框架**：Jetpack Compose (极简现代 Material 3 界面)
*   **数据存储**：Room 数据库 (安全演进无感迁移)
*   **网络实现**：Java/Kotlin 原生 Socket 直连服务 (无重型依赖，极度轻量与省电)
